# FitTrack · Entrenamiento funcional 6 meses

App para gestionar mi programa de entrenamiento: 11 entrenamientos (A–E, Casa A/B, 4 Extra KB), calendario semanal flexible, registro manual de pesos por ejercicio con doble progresión, y seguimiento de peso corporal, cintura y constancia.

**Todo está en un único archivo: `index.html`. No necesita instalación ni compilación.**

## Cómo publicarla (GitHub Pages, gratis)

1. Crea un repositorio nuevo en GitHub (público).
2. Pulsa **"uploading an existing file"** (o Add file → Upload files) y arrastra `index.html` y este `README.md`. Commit.
3. Ve a **Settings → Pages**.
4. En "Branch" elige **main** y carpeta **/ (root)**. Guarda.
5. En 1-2 minutos tu app estará en `https://TU_USUARIO.github.io/NOMBRE_DEL_REPO/`.

Ábrela en el móvil y usa **"Añadir a pantalla de inicio"** para tenerla como una app.

## Datos

Todo se guarda en el navegador (localStorage) bajo la clave `fitapp:v4`: pesos por ejercicio, entrenamientos completados, calendario, peso corporal, cintura y check-ins. Si usas siempre el mismo navegador del móvil, tus datos se conservan.

## Probarlo sin publicar

Haz doble clic en `index.html`: se abre directamente en cualquier navegador.
